By using the wildcard "*", or by configuring @CrossOrigin to use unsafe methods, such as PUT or DELETE, the application could become the target of code injection and/or other attacks. Spring by default only allows GET, HEAD, and POST methods.
Before:@CrossOrigin(methods = "*")
After:@CrossOrigin(methods = {RequestMethod.GET, RequestMethod.POST})
