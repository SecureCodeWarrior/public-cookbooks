Ensure the cookie is only sent over HTTPS by setting the setSecure() flag to true. Otherwise, when using the default setting, the cookie will be sent over an unencrypted HTTP connection and the session ID could be disclosed via a Man-in-the-Middle attack.
Before:Cookie cookie = new Cookie();
After:Cookie cookie = new Cookie(); cookie.setSecure(true);
