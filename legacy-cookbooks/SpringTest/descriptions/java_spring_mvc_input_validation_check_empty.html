Abstract

Secure coding practices prescribe to check input parameters for emptiness using encoding libraries.

Description

It is recommended to check user input by using encoding libraries provided by the framework. This will not only improve security but ensures the data you receive is as expected to be used in other processes.

Correct code example:

import org.springframework.validation.ValidationUtils;
...
ValidationUtils.rejectIfEmptyOrWhitespace(errors, username, "username.required");

Violating this guideline can cause

Web vulnerabilities
