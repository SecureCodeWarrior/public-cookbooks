Abstract

Secure coding practices prescribe to use the BCryptPasswordEncoder implementation for password encoding.

Description

The PasswordEncoder interface is a service interface for encoding passwords. The preferred implementation specified by the Spring Framework documentation is the BCryptPasswordEncoder. It is recommended to use the BCryptPasswordEncoder over other implementations.

Class information:

package org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder
   String encode(CharSequence rawPassword)
   boolean matches(CharSequence rawPassword, String encodedPassword)

Correct code example:

import org.springframework.security.crypto.password.PasswordEncoder;
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;
...
PasswordEncoder bcryptPwdEncoder = new BCryptPasswordEncoder();
String hashedPwd = bcryptPwdEncoder.encode(password);
