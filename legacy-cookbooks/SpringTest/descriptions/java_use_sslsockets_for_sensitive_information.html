Abstract

Secure coding practices prescribe that SSLSocket class should be used to communicate sensitive information.

Description

The communication between instances of the Socket class happens over plaintext, so this communication method does not offer any security guarantees. It is recommended to use an SSLSocketFactory to create SSLSockets. These sockets add a layer of security over the underlying network transport protocol to guarantee Integrity, Authentication and Confidentiality.

Correct code example:

import java.net.ssl.SSLSocketFactory;
...
Socket secureSocket = SSLSocketFactory.getDefault().createSocket(RECEIVER_IP, 443);
