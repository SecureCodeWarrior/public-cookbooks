Abstract

Secure coding practices prescribe that Denial of Service (DoS) prevention should be taken. There should always be a maximum waiting time.

Description

Endless loops or infinite waiting should be avoided by always setting a maximum waiting time.
A maximum timeout duration static constant SCWConstants.MAXTIMEOUTDURATION can be used to achieve this goal. Use such a constant by taking the minimum of the user provided value and the MAXTIMEOUTDURATION.

Correct code example:

long maxtime = Math.min(user_provided, SCWConstants.MAXTIMEOUTDURATION);
Thread.sleep(maxtime);
There should also be taken care around Thread.join() and infinite loops.
