Content Provider Types

Based on tags and attributes used in Content Providers, four types can be specified: private, public, partner and in-house.
  1. Private Content Provider: Content Provider cannot be used by other applications. 
    <provider android:exported="false" ... >
  2. Public content Provider: Content Provider can be used by any other applications. 
    <provider android:exported="true" ... >
  3. Partner Content Provider: Content Provider can be used by partner applications.
    Same as Public Content Provider, however, verify the certificate of requesting app in code.
  4. In-House Content Provider: Content provider can be used by in-house applications.
    Verify the In-House signature permission in code. 
    <provider android:exported="true"
   android:permission="com.senseissecurity.provider.MY_PERMISSION" />
