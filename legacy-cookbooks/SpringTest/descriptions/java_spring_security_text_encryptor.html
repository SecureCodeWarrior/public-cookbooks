Abstract

Secure coding practices prescribe to use a text encryptor that performs encryption.

Description

It is recommended to not use the noOpText text encryptor, but instead use an encryptor that actually performs encryption.

Class information:

package org.springframework.security.crypto.encrypt.Encryptors
    static TextEncryptor delux(CharSequence password, CharSequence salt)
    static TextEncryptor queryableText(CharSequence password, CharSequence salt)
    static TextEncryptor text(CharSequence password, CharSequence salt)

Correct code example:

import org.springframework.security.crypto.encrypt.Encryptors;
encryptors.text(password, salt);
