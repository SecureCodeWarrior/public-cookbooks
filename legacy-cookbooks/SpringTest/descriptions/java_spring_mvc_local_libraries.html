Abstract

Secure coding practices prescribe to only load libraries from local resources.

Description

It is recommended to load resources from the same security domain. Including third-party JavaScript libraries through HTTP is deemed insecure. For this reason only local resources should be used to load libraries.

Correct code example:

<script type="text/javascript" src="path/to/local_resource.js"></script>

Violating this guideline can cause

Web vulnerabilities
