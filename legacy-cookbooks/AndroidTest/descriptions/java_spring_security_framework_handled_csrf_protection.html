Abstract

Secure coding practices prescribe to let the Spring Framework handle CSRF protection.

Description

As of Spring Security 4.0, CSRF protection is enabled by default with XML and Java configurations. It is never recommended to disable CSRF protections.

Violating this guideline can cause

Web vulnerabilities
