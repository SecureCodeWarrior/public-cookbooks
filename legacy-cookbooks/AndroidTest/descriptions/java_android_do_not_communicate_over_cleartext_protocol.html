Abstract

Out of best practices and Android coding guidelines, recommendations were abstracted which state that communication should be done over encrypted channels, such as HTTPS.

Description

Set the android:usesCleartextTraffic attribute in your manifest file to false.

Correct code example:

android:usesCleartextTraffic="false"

Violating this guideline can cause

Mobile vulnerabilities

  • Insufficient Transport Layer Protection - Communication Over Cleartext Protocol. Learn more
