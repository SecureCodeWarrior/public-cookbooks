Deprecated Flag

FLAG_ACTIVITY_CLEAR_WHEN_TASK_RESET

This constant was deprecated in API level 7.
FLAG_ACTIVITY_CLEAR_WHEN_TASK_RESET does not work reliably in some versions of Android. This flag performs identically to FLAG_ACTIVITY_NEW_DOCUMENT which should be used instead of this.
