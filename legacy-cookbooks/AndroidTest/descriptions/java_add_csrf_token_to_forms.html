Abstract

Secure coding practices prescribe that every form should have a unique CSRF token.

Description

It is recommended to always add a hidden input-tag to the form-tag containing a unique CSRF token.

Correct code example of the input-tag that needs to be added to every form-tag:

<form method="POST" action="/transfer">
   ...
   <input type="hidden" name="csrftoken" value="{{csrftoken}}"/>
</form>

Violating this guideline can cause
