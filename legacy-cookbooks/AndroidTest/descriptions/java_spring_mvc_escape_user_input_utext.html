Abstract

Secure coding practices prescribe to not use th:utext.

Description

Using the Thymeleaf XML attribute th:utext allows for displaying unescaped text. Not escaping user input leaves the application vulnerable to XSS attacks. It is recommended to use th:text instead to automatically escape the text before displaying.

Violating this guideline can cause

Web vulnerabilities
