Abstract

Out of Android best practices and secure coding guidelines, recommendations were abstracted that state that meaningful constants should be used.

Description

To ensure good code quality it is recommended to use a meaningful constants instead of raw values. This makes to code more readable and makes it easier to identify possible coding errors or bugs.

Correct code example:

FileOutputStream fos = c.openFileOutput(file.getPath(), android.content.Context.MODE_PRIVATE);
