Abstract

Secure coding practices prescribe to allow the application to decide the channel to be used to send the form data.

Description

It is recommended to let the application sever decide the channel to be used to send form data. Through it's secure channel configurations it will make sure data is not exposed to an attacker sniffing the network. Forcing a form to submit over HTTP protocol (plain text) will leave confidential information vulnerable.

Violating this guideline can cause

Web vulnerabilities

  • Insufficient Transport Layer Protection - Unprotected Transport of Sensitive Information. Learn more
