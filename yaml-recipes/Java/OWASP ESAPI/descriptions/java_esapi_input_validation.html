Abstract

Secure coding practices prescribe that user input should always be sanitized and validated properly.

Description

User input should always be sanitized and validated properly before use. It is recommended to use a REGEX to validate user input.
The OWASP ESAPI validator can be used to achieve this.

Class information:

package org.owasp.esapi.Validator;

Correct code example:

import org.owasp.esapi.ESAPI;
...
if (!ESAPI.validator().isValidInput("Email", userBean.getEmail(), "Email", 70, false)) { 
   //handle invalid input
}
