It is recommended to define the HTTP request method. @RequestMapping has no default value to fall back on. If unspecified, it will map to any HTTP request, which could eventually lead to unwanted changes to the application.


Before 
@RequestMapping(path = "/delete_user")
After 
@DeleteMapping(path = "/delete_user")
