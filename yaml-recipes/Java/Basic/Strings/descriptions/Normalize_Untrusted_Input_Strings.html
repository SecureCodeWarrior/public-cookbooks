Normalize untrusted data before use.


Strings can contain different character sets. If unchecked, validation routines could be bypassed by using alternative representations of the same character (e.g. "\uFE64" for "<"). By applying the Normalizer.normalize method, we ensure that Unicode characters are converted to their normalized counterparts.


Before 
Matcher matcher = pattern.matcher(str);

After 
Matcher matcher = pattern.matcher(Normalizer.normalize(str, Form.NFKC));
