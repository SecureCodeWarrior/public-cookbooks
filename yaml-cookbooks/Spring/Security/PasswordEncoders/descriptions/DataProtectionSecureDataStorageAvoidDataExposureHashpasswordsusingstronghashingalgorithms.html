Abstract

Secure coding practices prescribe that passwords should be hashed using the BCryptPasswordEncoder.

Description

Passwords should be hashed using strong algorithms. The Spring framework provides this throught the BCryptPasswordEncoder class. 
org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder

Correct code example:

BCryptPasswordEncoder pwEncoder = new BCryptPasswordEncoder();
pwEncoder.encode(password);

Violating this guideline can cause

Web vulnerabilities

  • Sensitive Data Exposure
  • Plaintext Storage of Credentials
  • Use of Insecure/Deprecated Algorithms
