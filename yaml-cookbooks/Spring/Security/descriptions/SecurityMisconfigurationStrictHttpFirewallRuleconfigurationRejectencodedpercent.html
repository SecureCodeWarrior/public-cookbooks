Disallowing a URL encoded percent ("%25") in the URL is more secure, as they are frequently used to bypass security constraints. Spring Security by default sets StrictHttpFirewall#setAllowUrlEncodedPercent() to false.
Before:firewall.setAllowUrlEncodedPercent(true);
After:firewall.setAllowUrlEncodedPercent(false);
