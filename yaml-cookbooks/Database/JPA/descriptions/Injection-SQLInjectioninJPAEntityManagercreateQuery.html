Description

Dynamically creating queries by means of concatenating (untrusted) user input puts the application at risk of SQL injection. An attacker could insert malicious input to obtain or modify data. Use parameterized queries to prevent SQL injection. Before 
entityManager.createQuery("Select b from Books b where b.author like " + author).getResultList();
After 
entityManager.createQuery("Select b from Books b where b.author like :author")
.setParameter("author", author)
.getResultList();
