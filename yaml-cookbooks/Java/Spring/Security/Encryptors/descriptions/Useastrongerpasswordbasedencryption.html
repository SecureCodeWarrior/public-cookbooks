Encryptors.text() is a text encryptor that makes use of 'standard' password-based encryption. This mode is not authenticated, so confidentiality and authenticity of the data are never guaranteed. For more security, use Encryptors.delux() which implements the 'stronger' password-based encryption.


Before 
Encryptors.text(password, salt);
After 
Encryptors.delux(password, salt);
