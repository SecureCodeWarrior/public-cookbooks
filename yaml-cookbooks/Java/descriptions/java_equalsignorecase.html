Abstract

Secure coding practices prescribe that equalsIgnoreCase should be used to compare String case-insensitive.

Description

When comparing Strings it is not advised to convert them to all upper or all lower case Strings before comparing. Instead it is recommended to use equalsIgnoreCase.
The following method is available

Class information:

package java.lang.String;
public boolean equalsIgnoreCase(String);

Correct code example:

boolean correct = solution.equalsIgnoreCase(answer);
