Abstract

Out of Java best practices and coding guidelines, recommendations where abstracted that recommend the usage of OWASP Enterprise Security API (ESAPI). Routines from that library are recommended for session management. As such, it is discouraged to generate and handle account credentials or session identifiers using standard java routines.

Description

It is not recommended to use the standard java implementation for session management as there is an OWASP ESAPI alternative.
More specific, the guideline recommends that an implementation of the Authenticator interface should be used for generating and handling account credentials and session identifiers. Possible implementing classes of this interface are the AbstractAuthenticator and its extension FileBasedAuthenticator.
Other banned methods related to session management:
  • BAN004: Session.getId
No safe alternatives have been suggested by the ESAPI Guidelines.
