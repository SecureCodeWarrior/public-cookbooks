Abstract

Secure coding practices prescribe that all data going into webpages needs to be encoded properly.

Description

Find the context in which the data will be used and choose an appropriate encoder. It is recommended to use well-known encoders and avoid trying to remove illegal characters yourself.
For example the OWASP ESAPI Encoder can be used to achieve this.

Class information:

package org.owasp.esapi.Encoder;
   String encodeForCSS(String input);
   String encodeForHTML(String input);
   String encodeForHTMLAttribute(String input)
   String encodeForJavaScript(String input);
The encoder object has to be retrieved from the following class.

Class information:

package org.owasp.esapi.ESAPI;
   public static Encoder encoder();

Correct code example:

stringBuilder.append("<body> We cannot log you in ");
stringBuilder.append(ESAPI.encoder().encodeForHTML(username));
stringBuilder.append("</body>");
