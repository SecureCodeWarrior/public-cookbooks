Abstract

Out of Java best practices and coding guidelines, recommendations where abstracted that recommend the usage of OWASP Enterprise Security API (ESAPI). Running OS commands from the program should be avoided as much as possible as this is insecure. As such, in line with the ESAPI list of banned methods, it is discouraged to use any methods from the java.lang.Runtime class.

Description

It is discouraged to use the standard java implementation for running OS commands as they are insecure. It is best practice to avoid running OS commands from the program altogether.
