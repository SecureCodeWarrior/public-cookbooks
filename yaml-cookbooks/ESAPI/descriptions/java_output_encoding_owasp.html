Abstract

Secure coding practices prescribe that all data going into webpages needs to be encoded properly.

Description

Find the context in which the data will be used and choose an appropriate encoder. It is recommended to use well-known encoders and avoid trying to remove illegal characters yourself.
For example the OWASP Encoder can be used to achieve this.
