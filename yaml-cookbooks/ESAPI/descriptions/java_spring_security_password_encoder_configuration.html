Abstract

Secure coding practices prescribe to hash passwords in Java Spring Security using the sha-256 algorithm.

Description

It is recommended to use hash passwords with the strongest available hashing algorithm. In the Java Spring Security configuration files it is recommended to use sha-256.

Correct code example:

<security:authentication-manager>
   <security:authentication-provider>
      <security:password-encoder hash="sha-256" />
      <security:user-service>
         <security:user name="testuser" password="pasword123" authorities="ROLE_USER" />
      </security:user-service>
   </security:authentication-provider>
</security:authentication-manager>
