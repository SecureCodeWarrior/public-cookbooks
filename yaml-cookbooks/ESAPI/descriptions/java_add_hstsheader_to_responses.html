Abstract

Secure coding practices prescribe that the HTTP Strict Transport Security Header (HSTS) should be used in Http responses.

Description

When sending a response to a client the HSTS Header has to be added to the response. This allows the web server to advertise that they are willing to accept TLS-only clients. The following method can be used to add this header.

Class information:

package java.servlet.http.HttpServletResponse;
   public static void setHeader(String, String);
Apply this function with the correct HSTSHeader strings.

Correct code example:

response.setHeader("Strict-Transport-Security", "max-age=7776000; includeSubdomains")
response.senRedirect("sensei.com/some_page");
