Abstract

Out of Android best practices and secure coding guidelines, recommendations were abstracted for use of TLS/SSL.

Description

When using Transport Layer Security, it is encouraged to support TLS protocol version 1.2. Implementations must not negotiate TLS version 1.1 except when no higher version is available. Protocols SSLv2, SSLv3 and TLSv1.0 are prohibited because they are no longer deemed secure.
The following methods are available to ensure the right protocol version is used.

Class information:

package javax.net.ssl.SSLContext;
   public static SSLContext getInstance(String);

Correct code example:

// Support TLSv1.2.
public static SSLContext getInstance("TLSv1.2");

Violating this guideline can cause

Mobile vulnerabilities

  • Insufficient Transport Layer Protection - Weak Cipher Suites. Learn more
