Abstract

Out of best practices and android coding guidelines, recommendations were abstracted which state that android.media.ExifInterface should not be used.

Description

android.media.ExifInterface can lead to Denial of service attacks or code injection. Use android.support.media.ExifInterface instead.

Correct code example:

import android.support.media.ExifInterface;
In order to add this package to your dependencies add the folowing line of code to your gradle file: 
compile "com.android.support:exifinterface:25.1.0"

Violating this guideline can cause

Mobile vulnerabilities

  • Client Side Injection - JavaScript Injection. Learn more
