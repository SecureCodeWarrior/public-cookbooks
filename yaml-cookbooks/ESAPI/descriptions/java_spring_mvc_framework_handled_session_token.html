Abstract

Secure coding practices prescribe to let the framework or AppServer manage sessions.

Description

It is recommended to never send session IDs through URL parameters. Instead it is recommended to let the framework or AppServer manage sessions.

Violating this guideline can cause

Web vulnerabilities

  • Session Handling - Cross Site Request Forgery. Learn more
